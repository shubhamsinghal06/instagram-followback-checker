# Instagram Followback Checker

A simple privacy friendly tool to find accounts that do not follow you back on Instagram.

Works completely in the browser. No login required.

## How to use

1. Go to Instagram
2. Request your data export
3. Download the ZIP file
4. Extract it
5. Find these files:

followers_1.html or followers.json  
following.html or following.json

6. Upload them on the tool

## Deployment

Enable GitHub Pages and host the repository root.

## Privacy

All processing happens locally in your browser. No data is uploaded anywhere.
